Title: Web Store

Files

funcs.py - contains some fucntions regarding SQL requests, creating tables, and searching for different record along the database

products1.db - is the database, already contains all records needed for the application to work

main.py - pieces all together, also contains HTML code for all the pages needed

Description of the project

The goal of the project is to create a simple web store, connecting HTML pages and SQLite using Python. HTML pages, are contained in the main.py file

How to launch

Import all files in the same directory, launch main.py file. Using your internet browser of choise, go to localhost:8000
